Save Desktop

Save your desktop configuration

About

Save Desktop allows you to backup, restore, and synchronize your Linux desktop configuration, including themes, fonts, icons, wallpapers, extensions, desktop folder, Flatpak apps and their data, making it easy to maintain a consistent workspace across multiple devices.

It works on GNOME, KDE Plasma, Cinnamon, Budgie, Pantheon, COSMIC (Rust and GNOME version), Deepin, MATE, Xfce and Hyprland WM.

Actions


Save

Quickly create a new backup archive by choosing its name and selecting which items to include—such as themes, fonts, icons, wallpapers, extensions, desktop folder, and Flatpak apps with their data. You can also set up periodic automatic saving to ensure your workspace is always up to date.

Import

Import your desktop configuration from a ZIP archive or folder to instantly restore your themes, fonts, icons, wallpapers, extensions, desktop folder, and Flatpak apps with their data. Compatible with all supported desktop environments for seamless setup.

Sync

Set up automatic periodic backups and synchronize your desktop configuration with cloud storage. Keep your themes, fonts, icons, wallpapers, extensions, desktop folder, and Flatpak apps consistent across multiple devices. Learn more about synchronization here.
